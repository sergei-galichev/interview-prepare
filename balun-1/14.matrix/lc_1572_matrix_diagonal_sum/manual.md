[1572. Matrix Diagonal Sum](https://leetcode.com/problems/matrix-diagonal-sum/)

```go
package main

func diagonalSum(mat [][]int) int {
	sum := 0

	length := len(mat)

	for i := 0; i < length; i++ {
		// add main diagonal element
		sum += mat[i][i]

		// add sub diagonal element
		sum += mat[length-1-i][i]
	}

	if length%2 != 0 {
		halfIdx := length / 2

		sum -= mat[halfIdx][halfIdx]
	}

	return sum
}
```

***Оценка по времени:*** `O(n)`

*Объяснения:* Т.к. матрица квадратная размера `n x m` и `n` == `m`, то количество элементов, которое
пройдем, будет равно `n`

***Оценка по памяти:*** `O(1)`

*Объяснения:* дополнительной памяти выделяться не будет

**Описание решения**

Фактически нужно суммировать элементы, которые находятся на главной и побочной диагоналях матрицы. В том случае, если
матрица имеет нечетное количество строк (столбцов), то будет элемент, который находится на обеих диагоналях. Его нужно
учитывать только 1 раз. Для главной диагонали идем по элементам `m[i][i]`, для побочной - `m[size - 1 - i][i]`, где
`size` - размерность матрицы, `i` - индекс элемента
