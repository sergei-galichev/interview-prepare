[303. Range Sum Query - Immutable](https://leetcode.com/problems/range-sum-query-immutable/)

```go
package main

func countBits(n int) []int {
	result := make([]int, n+1)

	for i := 0; i <= n; i++ {
		count := 0

		num := i

		for num > 0 {

			count += num & 1

			num >>= 1
		}

		result[i] = count
	}

	return result
}
```

***Оценка по времени:***

`O(n * число бит в самом большом числе)`, где `n` - передаваемое максимальное целое число

*Объяснения:* фактически количество операций идти от 0 до передаваемого максимального целого числа и зависеть от коли-
чества значащих бит в нем

***Оценка по памяти:***

`O(1)`

*Объяснения:* дополнительная память в слайсе не выделяется

**Описание решения**

в цикле идем от `0` до переданного числа `n`. каждое число, кроме `0`, в цикле логически умножаем на `1` и результат
накапливаем в счетчике `count`. внутренний цикл идет пока число не станет `0`
